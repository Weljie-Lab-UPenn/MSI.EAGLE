# About MSI.EAGLE

MSI.EAGLE is an interactive platform for mass spectrometry imaging (MSI)
analysis. It supports standard MSI preprocessing, visualization, embedding,
clustering, and statistical testing, with optional multimodal registration and
single-cell polygon mapping workflows when needed.

Key features include:

- General MSI analysis workflows that do not require microscopy annotations
- Optional polygon-informed single-cell mapping into MSI `pData`
- Spatially aware UMAP and tissue-space visualization
- Histology, cluster-image, and polygon overlay registration
- Quantitative fit diagnostics and saved registration parameters
- Export-ready MSI datasets with mapped metadata
